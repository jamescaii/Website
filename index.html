MediCount

Used an Arduino and Raspberry Pi to build a device that monitors the pill bottle weight and detects weight changes in the pill bottle to indicate when pills have been taken. Designed to help people with any form of memory loss to remember whether or not they have taken their medication. Implemented using C++ and Python.

Demo Video Poster

Machine Mashups

Built a deep learning model that uses an attention based transformer network to create a song mashup given two input songs. Data was preprocessed by using a short-time fourier transform function to extract the magnitude signal and the phase signal, which was then placed into a 4D numpy array to be used by our model. The model consists of two parallel transformer-based LSTM architectures and was trained using mean squared error.

Spothouse

Developed a web application that allows users to create or join a room where each user has the ability to add their song to the queue and is able to vote on songs added by others. The song at the top of the queue is played through the host's Spotify account using the Spotify API. Implemented using Spark, React.js, and Java.
